# Додаток "Список покупок"

## Огляд проекту
Цей проект є додатком для управління списком покупок, створеним за допомогою FastAPI. Він дозволяє користувачам керувати своїми списками покупок, додаючи, видаляючи та оновлюючи елементи.

## Функціональні можливості
- Аутентифікація та авторизація користувачів
- Створення, читання, оновлення та видалення елементів списку покупок
- Захищені API-ендпоінти з використанням JWT-аутентифікації

# Документація API

## Точки доступу для автентифікації

### **POST /auth/**
**Створення користувача**
- **Опис:** Дозволяє новим користувачам зареєструвати обліковий запис.
- **Тіло запиту:**
  - `username` (рядок): Ім'я користувача для нового облікового запису.
  - `password` (рядок): Пароль для нового облікового запису.
  - `email` (рядок): Адреса електронної пошти користувача.
- **Відповідь:**
  - 201 Created: Користувача успішно створено.
  - 400 Bad Request: Помилки валідації.

### **POST /auth/token**
**Вхід для отримання токена доступу**
- **Опис:** Автентифікує користувача та повертає JWT-токен доступу.
- **Тіло запиту:**
  - `username` (рядок): Ім'я користувача.
  - `password` (рядок): Пароль користувача.
- **Відповідь:**
  - 200 OK: Токен успішно згенеровано.
  - 401 Unauthorized: Невірні облікові дані.

## Точки доступу для користувачів

### **GET /user/**
**Отримання користувача**
- **Опис:** Отримує дані автентифікованого користувача.
- **Відповідь:**
  - 200 OK: Дані користувача успішно отримані.
  - 401 Unauthorized: Користувач не автентифікований.

### **PUT /user/password**
**Зміна пароля**
- **Опис:** Дозволяє автентифікованому користувачу змінити пароль.
- **Тіло запиту:**
  - `old_password` (рядок): Поточний пароль користувача.
  - `new_password` (рядок): Новий пароль для встановлення.
- **Відповідь:**
  - 200 OK: Пароль успішно змінено.
  - 400 Bad Request: Помилки валідації або неправильний поточний пароль.
  - 401 Unauthorized: Користувач не автентифікований.

---

## Точки доступу для списків покупок

### **GET /shoppinglist/**
**Отримання списків покупок**
- **Опис:** Отримує всі списки покупок автентифікованого користувача.
- **Відповідь:**
  - 200 OK: Списки покупок успішно отримані.
  - 401 Unauthorized: Користувач не автентифікований.

### **POST /shoppinglist/**
**Створення списку покупок**
- **Опис:** Дозволяє користувачу створити новий список покупок.
- **Тіло запиту:**
  - `name` (рядок): Назва списку покупок.
  - `description` (рядок, необов'язкове): Опис списку покупок.
- **Відповідь:**
  - 201 Created: Список покупок успішно створено.
  - 400 Bad Request: Помилки валідації.
  - 401 Unauthorized: Користувач не автентифікований.

### **GET /shoppinglist/{list_name}**
**Отримання списку покупок**
- **Опис:** Отримує конкретний список покупок за назвою.
- **Відповідь:**
  - 200 OK: Дані списку покупок успішно отримані.
  - 404 Not Found: Список покупок не знайдено.
  - 401 Unauthorized: Користувач не автентифікований.

### **PUT /shoppinglist/{list_name}**
**Оновлення списку покупок**
- **Опис:** Оновлює дані існуючого списку покупок.
- **Тіло запиту:**
  - `name` (рядок, необов'язкове): Нова назва для списку покупок.
  - `description` (рядок, необов'язкове): Новий опис для списку покупок.
- **Відповідь:**
  - 200 OK: Список покупок успішно оновлено.
  - 404 Not Found: Список покупок не знайдено.
  - 401 Unauthorized: Користувач не автентифікований.

### **DELETE /shoppinglist/{list_name}**
**Видалення списку покупок**
- **Опис:** Видаляє список покупок за назвою.
- **Відповідь:**
  - 204 No Content: Список покупок успішно видалено.
  - 404 Not Found: Список покупок не знайдено.
  - 401 Unauthorized: Користувач не автентифікований.

## Точки доступу для елементів списків покупок

### **GET /shoppinglist/{list_id}/items**
**Отримання елементів для списку**
- **Опис:** Отримує всі елементи для конкретного списку покупок за його ID.
- **Відповідь:**
  - 200 OK: Список елементів успішно отримано.
  - 404 Not Found: Список покупок не знайдено.
  - 401 Unauthorized: Користувач не автентифікований.

### **POST /shoppinglist/{list_name}/item**
**Створення елемента**
- **Опис:** Додає новий елемент до конкретного списку покупок.
- **Тіло запиту:**
  - `name` (рядок): Назва елемента.
  - `quantity` (ціле число): Кількість елемента.
- **Відповідь:**
  - 201 Created: Елемент успішно додано.
  - 404 Not Found: Список покупок не знайдено.
  - 400 Bad Request: Помилки валідації.
  - 401 Unauthorized: Користувач не автентифікований.

### **PUT /shoppinglist/{list_name}/items/{item_name}**
**Оновлення елемента**
- **Опис:** Оновлює дані конкретного елемента у списку покупок.
- **Тіло запиту:**
  - `name` (рядок, необов'язкове): Нова назва для елемента.
  - `quantity` (ціле число, необов'язкове): Нова кількість для елемента.
- **Відповідь:**
  - 200 OK: Елемент успішно оновлено.
  - 404 Not Found: Список покупок або елемент не знайдено.
  - 400 Bad Request: Помилки валідації.
  - 401 Unauthorized: Користувач не автентифікований.

### **DELETE /shoppinglist/{list_name}/items/{item_name}**
**Видалення елемента**
- **Опис:** Видаляє конкретний елемент зі списку покупок.
- **Відповідь:**
  - 204 No Content: Елемент успішно видалено.
  - 404 Not Found: Список покупок або елемент не знайдено.
  - 401 Unauthorized: Користувач не автентифікований.
